# react에서는 데이터를 바꿀 수 있는 값이 변할때마다 실시간 UI를 변경해 달라고 알려야한다.
#  그럴때 사용하는것이 useState()이다.
```
const[num, setNum] = useState(0)

num = 변수처럼 사용되는 값
setNum = num 을 제어하는 함수
0 = num의 값
```
